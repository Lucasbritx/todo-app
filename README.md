# Todo app

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Lucasbritx/todo-app.git
```

Entre no diretório do projeto

```bash
  cd todo-app
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npx expo start
```

## Stack utilizada

**Front-end:** React Native, Styled-Components
